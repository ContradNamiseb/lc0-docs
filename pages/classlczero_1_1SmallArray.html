Detailed Description

template<typename T>
class lczero::SmallArray< T >

A lightweight, non-resizeable array implementation optimized for small collections.

This template class provides a simple array-like container that can hold up to 255 elements. It uses std::unique_ptr for automatic memory management and is optimized for small collections where dynamic resizing isn't needed. The size is limited to 255 elements (stored as unsigned char) to minimize memory overhead while providing sufficient capacity for small fixed-size collections commonly used in the chess engine.

Template Parameters
TThe type of elements stored in the array. Must be default-constructible.
