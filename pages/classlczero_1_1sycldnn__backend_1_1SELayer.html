Detailed Description

template<typename DataType>
class lczero::sycldnn_backend::SELayer< DataType >

Fused Squeeze-and-Excitation (SE) Layer.

Operations: (optional bias add +) global avg -> FC1 -> FC2 -> global scale -> add skip connection -> RELU.
