lczero::sycldnn_backend Namespace Reference

Implements the neural network backend using SYCL for GPU acceleration.

The sycldnn_backend leverages the SYCL abstraction layer to run Leela's neural network computations on a wide range of hardware, including Intel, AMD, and NVIDIA GPUs. It provides optimized implementations for key neural network layers such as:

  • Winograd Convolution: High-performance convolution for 3x3 filters (FusedWinogradConvSELayer).
  • Fully Connected Layers: Efficient matrix multiplication for the policy and value heads (FCLayer).
  • Squeeze-and-Excitation: Optimized kernels for the SE blocks (SELayer).
The backend manages memory and execution queues via the SyclContext and ensures thread-safe execution of concurrent search tasks.
