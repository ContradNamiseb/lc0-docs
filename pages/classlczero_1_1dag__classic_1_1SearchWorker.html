◆ ExecuteOneIteration()

void lczero::dag_classic::SearchWorker::ExecuteOneIteration ( )

Executes one full MCTS search iteration.

The iteration consists of the following steps:

  1. Initialize: Reset per-iteration workspace.
  2. Gather: Traverse the tree to find leaf nodes to expand (selection).
  3. Collision Check: Handle transpositions and potential collisions.
  4. Compute: Run the neural network inference on the gathered batch.
  5. Fetch: Retrieve results from the backend and populate new nodes.
  6. Backup: Propagate values (Q, N) up the tree to the root.
  7. Update: Update global search statistics and check stopping conditions.
