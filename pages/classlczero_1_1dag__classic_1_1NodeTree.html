Detailed Description

Manages the game tree (or DAG) for the search.

This class is responsible for the overall structure of the search tree, including the current root (head), the game history, and the transition between moves. It handles the logic for reusing the tree when moves are made or when the position is reset (transposition handling).

Resets the tree to a specific position defined by FEN and moves.

Attempts to reuse the existing tree structure if the new position is an extension or a known transposition of the current state. This preserves statistical information (visits, values) for the relevant nodes.

void lczero::dag_classic::NodeTree::TrimTreeAtHead ( )

Reinitializes the tree at the current head position.

Resets the current head to ensure it doesn't carry over details from a previous search.

Effectively disconnects the current head from previous history in terms of tree structure, allowing for a fresh search from this position without carrying over potential baggage from previous searches, while maintaining the position itself.
