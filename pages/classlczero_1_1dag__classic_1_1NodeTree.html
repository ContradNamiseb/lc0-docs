Detailed Description

Manages the game tree (or DAG) for the search.

The NodeTree class is the owner of the search graph. It maintains the root of the tree (gamebegin_node_), the current head of the search (current_head_), and the history of moves and positions. It is responsible for:

  • Advancing the tree when moves are played (MakeMove).
  • Resetting the tree to a new position, potentially reusing parts of the existing tree (ResetToPosition).
  • Trimming the tree to manage memory (TrimTreeAtHead).
