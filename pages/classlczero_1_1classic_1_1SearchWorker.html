Detailed Description

Single-threaded worker for MCTS search iterations.

Performs the core MCTS loop: node selection, expansion, evaluation, and backpropagation. Each SearchWorker runs in its own thread and coordinates with the main Search object for synchronization and stopping conditions. The worker handles minibatch gathering for efficient NN evaluation.

◆ ExecuteOneIteration()

void lczero::classic::SearchWorker::ExecuteOneIteration ( )

Executes one complete MCTS iteration.

Executes a single iteration of the search (initialization, gather, NN, backup).

The iteration consists of 7 phases:

  1. Initialize internal structures
  2. Gather minibatch of nodes to evaluate
  3. Prefetch positions into cache
  4. Run neural network computation
  5. Retrieve NN results into nodes
  6. Backpropagate values through tree
  7. Update search statistics
