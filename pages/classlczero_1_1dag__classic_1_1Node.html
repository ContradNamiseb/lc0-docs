lczero::dag_classic::Node Class Reference

Represents an expanded edge in the search DAG.

A Node corresponds to an edge in the graph that has been visited. It wraps an Edge (which contains the move and policy) and adds Monte Carlo statistics such as visit count (N), value (Q), and virtual loss.

Crucially, a Node links to a LowNode, which represents the resulting position. Multiple Nodes from different parents can point to the same LowNode, forming the DAG structure that handles transpositions.

Detailed Description

Represents an expanded edge in the search DAG.

A Node corresponds to an edge in the graph that has been visited. It wraps an Edge and adds Monte Carlo statistics (visits, value) and links to the resulting position (LowNode).

Nodes are the primary building blocks of the search tree. They are stored in a NodeTree and are managed by a NodeGarbageCollector to ensure thread safety during deletion.
