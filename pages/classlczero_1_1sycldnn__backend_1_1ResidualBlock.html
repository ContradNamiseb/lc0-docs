Detailed Description

template<typename DataType>
class lczero::sycldnn_backend::ResidualBlock< DataType >

Implements a Standard Residual Block using SYCL.

A Residual Block constitutes the main building block of the ResNet architecture used in Leela Chess Zero. It consists of two convolution layers with a skip connection adding the input to the output. Structure:

  • First Conv: A 3x3 convolution (typically Winograd-optimized) followed by Batch Normalization and ReLU activation.
  • Second Conv: Another 3x3 convolution followed by Batch Normalization.
  • Skip Connection: The input to the block is added to the output of the second convolution.
  • Final Activation: A ReLU activation is applied to the sum.
  • Optional SE: A Squeeze-and-Excitation block can be applied after the convolutions but before the residual addition.
This class manages the sub-layers (typically FusedWinogradConvSELayer) and coordinates their execution.
