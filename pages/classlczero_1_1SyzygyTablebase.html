Detailed Description

Interface for loading and probing Syzygy endgame tablebases.

Provides methods to initialize tablebases from disk and probe them for WDL/DTZ information. All probe methods are thread-safe once init() has completed; however, init() itself is not thread-safe and must not be called concurrently with any other method.

Usage:

SyzygyTablebase tb;
if (tb.init("/path/to/tablebases")) {
if (tb.max_cardinality() >= position.piece_count()) {
ProbeState result;
WDLScore wdl = tb.probe_wdl(position, &result);
if (result == ProbeState::OK) {
// Use wdl result
}
}
}
