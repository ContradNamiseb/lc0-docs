Detailed Description

Implementation of Syzygy tablebase probing.

This file contains the implementation for loading and probing Syzygy endgame tablebases. The original algorithm is based on Ronald de Man's Fathom/cfish implementation, adapted for use within Lc0.

The implementation supports:

  • WDL (Win/Draw/Loss) tables with suffix .rtbw
  • DTM (Distance To Mate) tables with suffix .rtbm
  • DTZ (Distance To Zeroing move) tables with suffix .rtbz

Key data structures:

  • PairsData: Huffman compression data for tablebase entries
  • EncInfo: Encoding information for piece/pawn positions
  • BaseEntry: Base class for tablebase entries
  • PieceEntry/PawnEntry: Specialized entries for piece vs pawn endgames
  • SyzygyTablebaseImpl: Internal implementation class
