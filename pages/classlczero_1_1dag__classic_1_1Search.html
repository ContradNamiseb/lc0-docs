Detailed Description

Main search engine for the DAG-based MCTS implementation.

This class orchestrates the search process using a Directed Acyclic Graph (DAG) structure to handle transpositions efficiently. It manages worker threads, neural network evaluations, time management, and UCI communication. Unlike the classic tree search, this implementation allows nodes to be shared across different paths (transpositions), reducing redundant work.
