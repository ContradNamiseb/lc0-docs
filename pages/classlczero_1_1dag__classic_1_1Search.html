lczero::dag_classic::Search Class Referenceabstract

#include <search.h>

Main search engine for the DAG-based MCTS implementation.

This class orchestrates the search process using a Directed Acyclic Graph (DAG) structure to handle transpositions efficiently. It manages worker threads, neural network evaluations, time management, and UCI communication.

Unlike the classic tree search, this implementation allows nodes to be shared across different paths (transpositions), reducing redundant work and producing more accurate evaluations for transposition-heavy positions.

Detailed Description

MCTS Search implementation for the dag_classic backend.

The Search class orchestrates the Monte Carlo Tree Search. It manages the search tree (NodeTree), the worker threads (SearchWorker), and the interaction with the neural network backend. It handles the iterative deepening, time management, and the selection of the best move.
