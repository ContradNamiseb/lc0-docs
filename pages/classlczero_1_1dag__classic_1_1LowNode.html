Detailed Description

Represents a unique position in the game DAG.

A LowNode corresponds to a specific board position. Unlike a Node, which represents an edge leading to a position, the LowNode holds the properties of the position itself, such as:

  • The list of legal moves (edges).
  • Neural network evaluation results (policy priors, value).
  • Reference count of parents (incoming edges) to detect transpositions.

If multiple paths lead to the same position (transposition), the corresponding Nodes all point to the same shared LowNode.
