Detailed Description

Abstract base class for all search algorithm implementations.

Defines the interface that all search algorithms (e.g., classic(vanilla MCTS), dag_classic) must implement. Provides virtual methods for setting up the search environment, controlling search execution, and retrieving results. The search is designed to be non-blocking where possible, allowing the UCI protocol to remain responsive during analysis.

Starts the search clock for time management.

Must not block; returns immediately. Can be called before or after StartSearch() depending on the timing mode:

  • Strict timing: called before SetPosition()
  • Normal mode: called before StartSearch()
  • Ponder mode: may be called at ponderhit
