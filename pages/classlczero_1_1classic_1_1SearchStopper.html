Detailed Description

Abstract base class for search stopping conditions.

Stoppers determine when the search should terminate based on various criteria (time, nodes, depth, etc.). Note:

  1. Stoppers are shared between threads - use synchronization for mutable state
  2. IterationStats and StoppersHints are per-thread, no synchronization needed
  3. OnSearchDone is called exactly once from a single thread
