Detailed Description

template<bool use_eigen>
class lczero::WinogradConvolution3< use_eigen >

Convolution 3x3 on a 8x8 board using the Winograd algorithm.

Implementation based on "Fast Algorithms for Convolutional Neural Networks".

Template Parameters
use_eigenIf true, uses Eigen; otherwise uses BLAS.
