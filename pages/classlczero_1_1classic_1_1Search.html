Detailed Description

Main MCTS search engine for the classic search implementation.

Orchestrates the Monte Carlo Tree Search, managing worker threads, neural network evaluations, time management, and UCI communication. The search operates on a NodeTree and coordinates multiple SearchWorker threads to explore the game tree in parallel.
