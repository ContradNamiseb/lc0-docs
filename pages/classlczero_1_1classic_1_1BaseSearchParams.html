Detailed Description

Base configuration parameters for MCTS search.

Contains all tunable parameters that control search behavior including CPUCT exploration constants, temperature settings, noise, FPU (First Play Urgency), collision handling, and WDL rescaling. Parameters are cached at construction for thread-safe access during search.
